Domain 1

This domain introduces the fundamentals of HTML (Hypertext Markup Language) and CSS (Cascading Style Sheets). It covers the basic structure of an HTML document, including the doctype, html, head, and body elements. Students learn about essential HTML elements for structuring content like headings, paragraphs, lists, and links. The domain also explains how to use metadata elements like title, meta, link, and style to provide information about the webpage. On the CSS side, students are taught how to style HTML content using selectors (element, class, id), properties, and values. They practice creating internal and external stylesheets and applying styles inline. The domain covers CSS syntax, rule sets, and the concept of the cascade (how styles are inherited and prioritized).

Domain 2

Building on the basics, this domain dives deeper into structuring HTML documents using semantic elements introduced in HTML5. Students learn to use elements like header, nav, main, article, section, aside, and footer to organize content in a logical and accessible manner. The domain also covers creating style sheets, understanding style sheet precedence (inline, internal, external), and writing valid CSS syntax. Students practice applying styles using classes, ids, and pseudo-classes like :hover. They learn techniques for separating structure (HTML) from presentation (CSS) for better maintainability.

Back to Top

Domain 3

This domain focuses on creating navigation systems and links within HTML documents. It covers different types of links (absolute, relative, bookmarks), as well as elements like a, map, and area for navigation. Students also learn about HTML tables, headings, paragraphs, lists, and semantic tags like details/summary and figure/caption. The domain introduces HTML forms and form controls like input, select, textarea, button, output, datalist, fieldset, and legend. Students practice implementing accessible forms with proper labels and field grouping.

Back to Top

Domain 4

This domain teaches how to embed multimedia content like images, videos, and audio into HTML documents. Students learn to use elements like img, video, audio, and iframe, as well as supporting elements like source and track for providing multiple media sources and captions. The domain also covers image layouts, creating image links, and potentially using canvas or SVG for graphics. Students apply CSS techniques to control the layout and positioning of multimedia elements on the page.

Back to Top

Domain 5

This extensive domain dives deep into using CSS for layout and positioning of page elements. It covers the CSS box model, including properties for setting height, width, margins, padding, and borders. Students learn different positioning techniques like static, relative, absolute, fixed, and float. The domain also explores concepts like overflow handling, display types (block vs inline), visibility, text formatting (fonts, colors, spacing, etc.), background properties, and CSS units and measurements. An important topic is creating responsive layouts using viewport units, media queries, CSS frameworks/grids, and flexible units like percentages.

Back to Top

Domain 6

The final domain emphasizes building accessible and user-friendly websites that work across different browsers and devices. It covers accessibility principles like providing text alternatives for non-text content, sufficient color contrast, logical tab order, and resizable text. Students learn techniques for improving code maintainability, such as reusing CSS rules, commenting code, and separating structure from presentation. The domain also stresses the importance of cross-browser testing to identify and fix rendering issues and syntax errors across different user agents.
